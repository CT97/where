
位置追踪 这是一个关于地图定位的项目,实现原理是APP去定时(每10分钟或更少)通过手机GPS定 位收集手机的当前位置地理坐标,然后上传至云数据库,手机之间通过帐号密码进行人物关联 ,比如一家四口就可以创建一个组,只有组之间才能进行位置共享. 设计此应用的目的在于家人之间旅游,出差等可以确定对方的位置,现在新闻中经常报 道女大学生失踪等案件,通过此应用即可查找对方手机的所在位置,然后进行求助帮助.
